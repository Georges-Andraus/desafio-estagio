Meu projeto é um sistema CRUD básico para cadastro de profissionais de saúde utilizando PHP e o framework Yii2, no qual a conexão com banco de dados (MySQL) e servidor Apache implementado via XAMPP.


REQUIREMENTS
------------
• Xampp
• PHP 8.1
• Framework Yii2
• MySQL
